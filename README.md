
# 📝 Руководство по написанию сценариев (.story)

## 📑 1. Базовый синтаксис

Сценарий выполняется последовательно, блок за блоком. Вся логика завязана на **Действиях (Метках)** и **Командах**.

### 🛠️ Основные правила

  <tr>
    <th width="20%">Элемент</th>
    <th width="50%">Описание</th>
    <th width="30%">Пример</th>

  <tr>
    <td><code>Метка:</code></td>

> **Правило одного блока:** 1 блок в сценарии — это 1 шаг игрока. **Не пишите** в одном блоке несколько команд `Show:`, так как они выполнятся одновременно без задержек. Блоки разделяются в рантайме нажатием игрока на Пробел или Клик мыши.

## 📦 2. Работа с ассетами (Ресурсами)

Движок поддерживает два источника загрузки графики: встроенные ресурсы игры и кастомные папки модов.

* **Встроенные ассеты игры (Запечённые):**
  Путь всегда должен строго начинаться с `Assets/`. Расширение файла писать не нужно.
  ```text
  Background: Assets/Backgrounds/ext_square_day
  ```

* **Внешние ассеты (Из папки вашего мода):**
Формат: `[id_мода]/[любая_папка]/[имя_файла]`. Движок сам найдет файл в папке вашего мода и подгрузит его. Расширение файла указывать необязательно.
  ```text
  Background: my_cool_mod/BG/forest_rain
  ```

## 💬 3. Диалоги и Текст

* **Показать текст / Диалог:** `Show: "Имя/HEX-Цвет" -> "Текст"`
```text
Show: "Алиса/#FF8900FF" -> "Привет, $(player_name)!"
ИЛИ
Show: "Рассказчик/#FFFFFFFF" -> "Вокруг стояла гробовая тишина..."
```


* **Изменение скорости печати текста:** `ChangeTextSpeed -> Число`
```text
ChangeTextSpeed -> 50
```



### 🔮 Кастомные теги для текста (TextMeshPro)

Вы можете оборачивать слова в тексте в специальные теги для создания невероятных эффектов:

* `<shake>` — нервная тряска букв.
* `<rainbow>` — текст переливается радугой и движется мягкой волной.
* `<bounce>` — буквы весело подпрыгивают.
* `<glitch>` — эффект цифрового сбоя (киберпанк).
* `<chaos>` — буквы хаотично сжимаются, растягиваются и наклоняются.
* `<ghost>` — буквы плавно растворяются, улетая вверх, и циклично появляются снизу.
* `<matrix>` — бегущая волна ярко-зелёного свечения компьютерного кода.
* `<drunk>` — буквы плавно покачиваются по овальной орбите.
* `<pulse>` — пульсирующее увеличение и уменьшение размера букв.

## 🔀 4. Логика, Переменные и Переходы

* **Вывод переменных в тексте:** Используйте конструкцию `$(имя_переменной)`.
```text
Show: "Интерфейс" -> "У вас в кармане ровно $(money) рублей."
```

* **Мгновенный переход:** `GoTo: имя_метки` — перепрыгивает на блок сразу во время выполнения текущего шага.
* **Отложенный переход:** `GoToDelayed: имя_метки` — перейдет на блок только после того, как игрок кликнет по текущему блоку текста.
* **Переход с возвратом (Функция):** `Call: имя_метки` — GoTo, но с возвратом обратно. Выполняется, после чего продолжает выполнять свой блок далее.
* **Управление переменными:**
```text
SetVar: money -> 0        # Создать/установить число (Int)
AddInt: money -> 100      # Прибавить 100 к переменной
AddInt: money -> -50      # Отнять 50 из переменной
SetBool: met_alice -> true # Установить логический флаг (true/false)
```

* **Принудительное завершение:** `Exit` — мгновенно прерывает сценарий, выгружает мод и возвращает игрока в Главное Меню.

## ⚖️ 5. Условные блоки (Ветвление)

> **Правило прыжка:** Если условие **выполняется**, движок выполняет команду на следующей строке. Если условие **ложно**, движок перепрыгивает через одну строчку.

* **Проверка чисел (`IfInt`):** `IfInt: переменная -> Условие -> Число`
  <br>*Условия:* `Less`, `LessEqual`, `Equal`, `GreaterEqual`, `Greater`, `NotEqual`
```text
IfInt: money -> GreaterEqual -> 100
    GoTo: buy_item
Show: "Продавец" -> "У вас недостаточно денег!"
```

* **Проверка флага (`IfBool`):** `IfBool: переменная -> Условие -> true/false`
*Условия:* `Equal`, `NotEqual`
```text
IfBool: met_dog -> Equal -> true
    Show: "Собака" -> "Гав! Гав!"
```

## 👥 6. Управление персонажами

* **Добавить персонажа:**
```text
Character: alice_id ->
    Sprite -> Assets/Characters/Alice/Normal
    Position -> 0
```

*(Координата `Position` настраивается в диапазоне от `-15` до `15`)*
* **Передвинуть персонажа:**
```text
MoveCharacter: alice_id ->
    Position -> -5.5
    MoveType -> Linear
    MoveSpeed -> 4.5
```

*Доступные типы движения (`MoveType`):* `Momental` (телепорт), `Linear` (плавно), `Lerped` (по плавной кривой затухания).
* **Сменить эмоцию/спрайт:** `CharacterEmotion: ID -> "Путь_к_спрайту"`
```text
CharacterEmotion: alice_id -> "Assets/Characters/Alice/Sad"
```

* **Убрать со сцены:** `RemoveCharacter: ID`
```text
RemoveCharacter: alice_id
```

## 🎬 7. Эффекты камеры

Команда запускает эффекты кинематографичной тряски камеры. Эффект выполняется параллельно тексту и не блокирует чтение диалогов.

Синтаксис поддерживает два формата: **быстрые пресеты** и **ручную настройку**.

### А) Использование готовых пресетов

Формат: `CameraEffect: [Тип_Тряски] -> [Имя_Пресета]`
<br>*Доступные типы:* `HorizontalShake` (вправо-влево), `VerticalShake` (вверх-вниз).

```text
# Короткий мощный удар
CameraEffect: VerticalShake -> Punch

# Затяжное и сильное землетрясение
CameraEffect: HorizontalShake -> Earthquake
```

| Имя Пресета | Описание | Параметры (Сила; Частота; Время) |
| --- | --- | --- |
| `HeavyShortShake` | Тяжелый резкий взрыв | `1.0; 50.0; 0.3` |
| `LightShortShake` | Легкий секундный испуг | `0.3; 50.0; 0.3` |
| `Punch` | Физический удар / Столкновение | `0.5; 50.0; 0.2` |
| `Explosion` | Оглушительный взрыв неподалеку | `0.5; 37.5; 2.0` |
| `Earthquake` | Мощный гул и тряска земли | `0.4; 15.0; 6.0` |
| `Paranoia` | Легкий тремор экрана (паника/страх) | `0.08; 5.0; 10.0` |

### Б) Кастомная точная настройка

Если вам нужен уникальный эффект, передайте параметры вручную через стрелочки:
Формат: `CameraEffect: [Тип] -> [Амплитуда] -> [Частота] -> [Время угасания]`

```text
# Трясти камеру вверх-вниз; амплитуда 0.8; частота 60; в течение 1.5 секунд
CameraEffect: VerticalShake -> 0.8 -> 60 -> 1.5
```

## 🔉 9. Звуки
> [!WARNING]
> Лучше всего использовать .ogg для больших звуков, но есть поддержка .mp3 и .wav файлов. При загрузке больших эмбиентов игра может зависнуть на время прогрузки звука, использовать большие эмбиенты нежелательно, но не запрещено.

Настройка звука состоит из трех параметров:

> **Панорама** (от -1.0 до 1.0) — определяет, с какой стороны игрок услышит звук:<br>
  ***-1.0*** = звук полностью слева<br>
  ***0.0*** = звук строго по центру<br>
  ***1.0*** = звук полностью справа

> **Дистанция** (от 0.0 до 5.0) — определяет удаление источника от игрока:<br>
  ***0.0*** = звук играет прямо "в голове" у игрока (максимальная громкость)<br>
  ***3.0*** = среднее расстояние (звук будет отчетливо приглушен)<br>
  ***5.0*** = максимальное расстояние (звук практически не слышно)

> **Громкость** (от 0.0 до 1.0) — общая громкость аудиофайла:<br>
  ***0.0*** = полная тишина<br>
  ***1.0*** = оригинальная громкость файла (100%)

* **One-shot** (реплики, звуки интерфейса, всё что должно проиграться один раз):

  ```text
  PlaySound: путь_к_файлу -> панорама -> дистанция -> громкость
  ```
  Примеры:
  ```text
  # Озвучка реплики персонажа (в центре)
  PlaySound: Assets/Voiceovers/VO_1 -> 0 -> 1 -> 1
  # Звук стука в дверь (слева)
  PlaySound: Assets/Sounds/DoorKnock -> -1 -> 3 -> 0.8
  ```

* **Ambient** (эмбиенты, саундтреки):<br>
  При запуске, прошлый эмбиент медленно затухает, а новый медленно набирает громкость.<br>
  Выбрать канал пока нельзя, будет пофикшено в следующих обновлениях.
  ```text
  PlayAmbient: путь_к_файлу -> скорость_затухания
  ```
  Примеры:
  ```text
  # Запуск эмбиента, скорость затухания прошлого эмбиента - 5 секунд.
  PlaySound: Assets/Ambients/Ambient1 -> 5
  # Запуск саундтрека, который заменит эмбиент.
  PlaySound: Assets/Soundtrack/Track1 -> 10
  ```

## 🚀 10. Полный пример готового скрипта

```text
init:
    ChangeTextSpeed -> 3500
    SetVar: money -> 0
    
    # Спавним Алису в центре экрана
    Character: alice ->
        Sprite -> Assets/Characters/Alice/Normal
        Position -> 0
        
    Background: Assets/Backgrounds/ext_square_day
    GoTo: start

start:
    Show: "Алиса/#FF8900FF" -> "Привет! Прямо сейчас у вас <color=orange>$(money)</color> монет."
    GoToDelayed: choice_zone

choice_zone:
    ShowChoiceButton:
        "Дать Алисе 100 золотых" -> "give_money"
        "Отобрать у Алисы 100 золотых" -> "steal_money"

give_money:
    AddInt: money -> -100
    CharacterEmotion: alice -> "Assets/Characters/Alice/Grin"
    
    # Камера сотрясается от радости, Алиса отбегает влево
    CameraEffect: HorizontalShake -> LightShortShake
    MoveCharacter: alice ->
        Position -> -15
        MoveType -> Lerped
        MoveSpeed -> 3
        
    Show: "Алиса/#FF8900FF" -> "Ого, спасибо огромное! Ты такой щедрый!"
    GoToDelayed: quit

steal_money:
    AddInt: money -> 100
    CharacterEmotion: alice -> "Assets/Characters/Alice/Sad"
    
    # Жесткий удар по камере
    CameraEffect: VerticalShake -> Punch
    MoveCharacter: alice ->
        Position -> 15
        MoveType -> Lerped
        MoveSpeed -> 3
        
    Show: "Алиса/#FF8900FF" -> "<shake>Эй!</shake> За что?! Это были мои последние сбережения..."
    GoToDelayed: quit

quit:
    Exit
```
