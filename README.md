## Гайд по написанию ".story" файлов
### Базовый синтаксис
1. `Код:` (в начале строки) — Создание нового действия (аналогично функциям), на них можно перейти через команды.
2. `Команда -> Значение` — Передача параметра в команду. Используется, когда нужно передать параметр напрямую в команду (смена скорости, запись в переменные и тд)
3. `Команда: Значение` — Использование параметра. Используется, когда нужно передать параметр на метки, фон или различные обьекты.
4. `Многострочные блоки` — Название пишется с одним табом, а все его свойства перечисляются ниже с двумя табами `Свойство -> Значение`

### 1. Диалоги / Настройки
* **Показать текст**: `Show: "Имя/HEX-Цвет"`
	Пример: `Show: "Алиса/#FF8900FF" -> "Привет, $(player_name)!"`
* **Изменение скорости печати текста**: `ChangeTextSpeed -> Число`
	Пример: `ChangeTextSpeed -> 5000`
* **Смена фона**: `Background: имя_файла`
	Пример: `Background: ext_square_day`

### 2. Логика / Переменные
* **Использование переменных в тексте:** `$(variable)`
	Пример: `У вас $(variable) денег`
* **Переход к другой сцене**: `GoTo: имя_сцены`
	Пример: `GoTo: main`
* **Создание/установка числовой переменной**: `SetVar: имя -> число`
	Пример: `SetVar: money -> 0`
* **Математические операции с переменной**: `AddInt: имя -> число`
	Поддерживает положительные и отрицательные числа.
	Пример: `AddInt: money -> 100` или `AddInt: money -> -100`
* **Установка флага (Логическая переменная)**: `SetBool: имя -> true/false`
	Пример: `SetBool: met_character -> true`

### 3. Условные блоки
**Правило работы:** если условие выполнено - выполняется следующий блок, если нет - движок перепрыгнет через 1 блок и выполнит его.
* **Проверка числа:** `IfInt: имя_переменной -> Условие -> Число`
	**Условия:** `Less`, `LessEqual`, `Equal`, `GreaterEqual`, `Greater`, `NotEqual`
	Пример:
	```
	IfInt: money -> GreaterEqual -> 100
		GoTo: buy_item		# Сработает, если выполнено условие
	Show: "Продавец/#FFFFFFFF" -> "У вас нет денег!" # Сработает, если денег меньше 100
	```
* **Проверка флага:** `IfBool: имя_переменной -> Условие -> true/false`
	**Условия:** `Equal`, `NotEqual`
	Пример:
	```
	IfBool: met_dog -> Equal -> true
		Show: "Сабака/#FFFFFFFF" -> "гаф"
	```

### 4. Управление персонажами на сцене
* **Добавить нового персонажа на сцену:**
	```
	Character: ID_Обьекта ->
		Sprite -> ИмяПерсонажа/ИмяФайла
		Position -> Число от -15 до 15
	```
* **Передвинуть персонажа:**
	```
	MoveCharacter: ID_Обьекта ->
		Position -> Число от -15 до 15
		MoveType -> Momental / Linear / Lerped
			(Моментальное, Линейное, По кривой)
		MoveSpeed -> Число
			(Только для линейного и по кривой)
	```
* **Изменить спрайт / эмоцию:** `CharacterEmotion: ID_Обьекта -> "ИмяПерсонажа/ИмяФайла"`
	Пример: `CharacterEmotion: character -> "Character/Sad"
* **Убрать персонажа:** `RemoveCharacter: ID_Обьекта`
	Пример: `RemoveCharacter: character`

### 5. Создание выбора (кнопок)
При создании окно открывается автоматически, а игра не продолжится, пока игрок не нажмёт на кнопку.
```
ShowChoiceButton:
	"Текст первой кнопки" -> "ID сцены для перехода"
	"Текст второй кнопки" -> "ID сцены для перехода"
```

### 6. Кастомные теги для текста
* `<shake>` - тряска текста
* `<rainbow>` - текст переливается радугой и двигается волной
* `<bounce>` - текст отпрыгивает
* `<glitch>` - эффект глитча из киберпанка
* `<chaos>` - буквы хаотично сжимаются, растягиваются и наклоняются в разные стороны
* `<ghost>` - буквы плавно растворяются, улетая вверх, а потом циклично появляются снизу
* `<matrix>` - бегущая волная ярко-зелёного свечения
* `<drunk>` - буквы плавно кружатся по овальной орбите
* `<pulse>` - пульсирующее увеличение и уменьшение размера букв
